# DDD-AIGC

本项目是作者的新书《悟道领域驱动设计》的随书案例代码。《悟道领域驱动设计》目前已上架各大电商平台，感兴趣的朋友可以帮忙支持一下。

<img alt="悟道领域驱动设计" src="assets/book.png" style="width: 50%">

购买链接：https://item.jd.com/14835288.html

本项目实现了一个贴纸日记的AIGC应用。

当用户需要记日记时，在一张贴纸上写下事情和参与者，系统会用ChatGPT生成日记正文。
