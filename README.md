# DDD-AIGC

本项目是《悟道领域驱动设计》（英文名：Thinking in Domain Driven Design）随书的案例代码。

《悟道领域驱动设计》在线电子书地址：

```text
https://github.com/feiniaojin/Thinking-in-DDD
```

本项目实现了一个贴纸日记的AIGC应用。

当用户需要记日记时，在一张贴纸上写下事情和参与者，系统会用ChatGPT生成日记正文。
